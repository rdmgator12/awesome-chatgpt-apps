# Contributing

This list tracks every app in OpenAI's official [ChatGPT Apps Directory](https://chatgpt.com/apps). Contributions welcome.

> This is an independent, community-maintained project. Not affiliated with, endorsed by, or sponsored by OpenAI.

## What You Can Contribute

### New Apps
When OpenAI adds new apps to the directory, submit a PR adding them to the appropriate category with a description and use case.

### Improved Descriptions
If a description or use case is missing detail or could be more helpful, submit a PR with a better one.

### Category Corrections
If an app is in the wrong category, submit a PR moving it.

### URL Corrections
The initial seed used best-known homepage URLs for each app. If a link is wrong, points to a competitor, or should redirect to a more specific landing page, submit a PR with the fix.

### Field Reports
Tested an app and have real-world notes? Add a brief field report below the app entry:

```markdown
- [App Name](https://example.com) - Description. *Use case: ...*
  > **Field report:** One paragraph on what worked, what didn't, what surprised you. Be specific.
```

## Guidelines

- One PR per change unless closely related.
- Keep descriptions concise — one sentence for the description, one for the use case.
- Use cases should be specific and practical, not marketing copy.
- Don't add apps that aren't in the official ChatGPT Apps directory. This list tracks the official directory, not all Apps SDK / MCP servers (see [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) for a broader MCP catalog).
- Maintain alphabetical order within categories.
- For apps that fit more than one category, pick the primary use and list once. Cross-references are fine in prose but not as duplicate entries.

## Weekly Updates

This list is updated weekly to stay in sync with the official directory. If you notice the directory has added apps that aren't listed here, please open an issue or PR.
